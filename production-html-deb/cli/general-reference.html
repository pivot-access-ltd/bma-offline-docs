General reference

This section provides general reference information about MAAS.

  • Release notes: These are general release notes for the latest version of MAAS, with links to release notes for older versions.

  • Installation requirements: MAAS has some minimal installation requirements that you’ll want to follow.

  • MAAS settings: Many things about MAAS can be customised.

  • MAAS source code: If you’re interested in how MAAS works, the source code is available.

  • Documentation style guide: Our documentation is crowd-sourced, but we don’t want it to look like it was built by random passers-by.

  • Glossary: MAAS has a fair amount of special terminology that you’ll want to know.

  • Our code of conduct: Canonical has a code of conduct, which we adhere to on the MAAS project.
