How to contact us

We would love to get your feedback on MAAS and how you are using it. You can get in touch with the team in two ways:

  1. The MAAS Discourse forum
  2. The MAAS Launchpad source code and issue tracking repository

If you’re interested in adopting MAAS for enterprise use or are looking to partner with Canonical, fill out the Contact Canonical form and someone will be in touch within one working day.

Alternatively, if you need some professional support, visit the Ubuntu Advantage shop.

Finally, if you find a problem with the documentation you can create an issue or contribute directly within the Docs Discourse category (the source of the documentation).
