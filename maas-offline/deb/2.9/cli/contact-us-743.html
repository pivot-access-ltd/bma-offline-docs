Contact us

We would love to get your feedback on MAAS and how you’re using it. You can get in touch with the team in various ways:

  • The MAAS IRC channel (external link) on freenode
  • The MAAS Discourse forum (external link)
  • The MAAS Launchpad (external link) source code and issue tracking repository

If you’re interested in adopting MAAS for enterprise use or are looking to partner with Canonical, fill out the Contact Canonical (external link) form and someone will be in touch within one working day.

Alternatively, if you need some professional support, visit the Ubuntu Advantage shop (external link).

Finally, if you find a problem with the documentation you can create an issue (external link) or contribute directly within the Docs Discourse category (external link) (the source of the documentation).
